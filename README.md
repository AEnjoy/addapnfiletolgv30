# LGApnVoLTEFile

基于Magisk模块,为LG v30 补全APN文件,并启用volte,dts功能

该模块将添加以下特性

> **三网APN**
>
> **VoLte的build.prop键值**
>
> DTS X 3D 音效开关

安装注意事项:

安装前请先确认你的设备是否为美版/欧版/日刷美/加版公开/美运营商版等(韩版未测试)

关闭对build.prop system less 的模块(或者在TWRP下刷入该模块),以免不生效

TWRP下刷入不可安装LG工具箱.

TWRP下若userdata未解密,则不会copy volte教程文件.

需要Magisk19.0+ 和Android O+
